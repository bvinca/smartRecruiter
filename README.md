# SmartRecruiter

SmartRecruiter is an intelligent Applicant Tracking System (ATS) that combines AI, NLP, and Retrieval-Augmented Generation (RAG) to automate recruitment tasks, rank candidates, and assist recruiters in making fair and informed hiring decisions.

## Features
- FastAPI backend for AI-driven applicant management
- React frontend dashboard for recruiters
- NLP + RAG modules for CV parsing and semantic ranking
- Dockerized setup for consistency
